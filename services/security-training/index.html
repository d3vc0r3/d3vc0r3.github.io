資安教育訓練
最完善的防禦，從了解攻擊開始

駭客技術及思維方式是開發者、系統管理者難以想像的，經由 DEVCORE 戴夫寇爾專業團隊的教育訓練，企業可以在事前防禦規劃上事半功倍，也更能在真實資安事件發生時，快速判定攻擊的目的、手法、漏洞，以進行應變處理

認知正確，才能擬定正確策略、正確應變

  • 協助企業系統化學習駭客攻擊思維與各種資訊安全觀念，從攻擊本質思考防禦策略，並提升開發者與管理者資安水平，強化企業整體資安體質及觀念
  • DEVCORE 資安教育訓練有別於坊間資安課程，包含各種入侵攻擊案例的線上環境，供學員實際操作練習
  • 針對攻擊概念、入侵步驟來說明有效的防禦之道，使學員可藉由實際操作及專業說明，瞭解如何面對駭客，從根源降低企業遭受損失的機率

為何企業需要資安教育訓練？
駭客的技術及思維方式是開發者、系統管理者難以想像的，經由 DEVCORE 的專業團隊教育訓練，企業可以在事前的防禦規劃上事半功倍，也更能在真實資安事件發生時，快速判定攻擊的目的、手法、漏洞，以進行應變處理。

課程列表

資安趨勢及最新攻擊案例

最新資安趨勢、最新資安漏洞與當前常見駭客入侵手法。

作業系統與網站安全攻防實務

網站攻防、伺服器攻防的理論與實際案例，使學員瞭解資訊系統全盤資安規劃。

網頁應用程式安全攻防實務

各種網站攻防概念以及滲透測試技巧與規範，使學員瞭解最核心的網站安全議題。

內網滲透安全攻防實務

常見提權手法與防範方法、內網橫向移動的思路及應對措施。

Microsoft Windows 伺服器系統安全

如何建置安全的 Windows 伺服器，並學習如何分析日誌與檢測惡意程式。

Linux 伺服器管理及安全實務

如何建置安全的 Linux 伺服器，並學習如何調整各種細部 Linux 安全設定。

社交工程偵蒐技巧

基於有限身份識別的目標關聯方法、知名社群平台隱私資訊汲取方法及模擬信件及語音型釣魚情境剖析。

常見 WAF、CDN 保護機制繞過實戰

教導異節點架構攻擊技術介紹、軟體及雲 WAF 繞過實例演練與探討以及路由規則（Routing Rule）限制繞過解析。

除了上述課程，亦可根據企業需求提供客製化課程。

為何選擇 DEVCORE 戴夫寇爾資安教育訓練？

DEVCORE 資安教育訓練有別於坊間資安課程，涵蓋各種入侵攻擊案例的線上環境，供學員上機操作練習；並由專家講解攻擊概念、入侵步驟與有效的防禦之道，使學員可藉由實際操作及專業說明，瞭解如何面對駭客，從根源降低企業遭受損失的機率。

瞭解真實駭客攻防技術的課程

DEVCORE 資安專家熟悉各種駭客社群發佈的最新情資及攻擊手法，因此得以將最真實的攻防概念融入課程中。知己知彼，百戰不殆，唯有理解攻擊者的手法、思維，企業才能建構更正確、完整的防禦機制。

涵蓋實機操作的訓練課程

企業透過實機操作、體驗駭客攻擊手法後，更能深刻理解如何建置安全的防護網。

適性、容易理解的訓練教材

資安技術長期演進改變，DEVCORE 以深厚技術背景為基礎，為客戶量身製作合適程度、易於吸收理解的教材，大幅提升教育訓練效率。
